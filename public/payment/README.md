# Ödeme logoları

`iyzico-logo-band.svg` (ve `@2x.png` sürümü), iyzico'nun **resmî logo paketindeki**
footer logo bandıdır. Tek görselde şunları içerir:

iyzico ile Öde · Mastercard · Visa · American Express · Troy

Kaynak: iyzico logo paketi → `footer_iyzico_ile_ode/Colored/logo_band_colored.svg`
(https://docs.iyzico.com/ek-bilgiler/iyzico-logo-paketi)

`components/marketing/Footer.tsx` bu bandı kullanır. Koyu zeminli bir footer'a
geçilirse paketteki `White` (beyaz) sürümle değiştirin.
